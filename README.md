# Спросить у Алисы — Browser Extension

Расширение для браузера, которое добавляет быстрый доступ к Yandex Alice AI через текстовый ввод.

## Как это работает

1. Нажмите Ctrl+Shift+A (или кликните по иконке расширения)
2. Введите текстовый запрос
3. Нажмите Enter — откроется новая вкладка с Алисой и ответом
