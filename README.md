# Спросить у Алисы — Browser Extension v2.0

Расширение для браузера, которое добавляет быстрый доступ к **Yandex Alice AI** через текстовый ввод — **с ответом прямо в popup**, без переключения вкладок.

## Возможности

### ✨ Главная фича: ответ в popup (#6)
Введите вопрос → Алиса отвечает прямо в popup-окне. Не нужно открывать новую вкладку.

Технически: расширение открывает **скрытую вкладку** (невидимую для вас), content script автоматически отправляет запрос, извлекает ответ из DOM и передаёт обратно. Скрытая вкладка тут же закрывается.

### 📜 История запросов (#2)
Последние 20 запросов сохраняются. Можно:
- Кликнуть по истории — запрос заполнится в поле ввода
- Удалить отдельный запрос (×)
- Свернуть/развернуть историю

### 🔍 Быстрый ввод из адресной строки (#7)
В адресной строке браузера введите:
```
alice погода москва
```
И нажмите Enter — сразу откроется (или переиспользуется) вкладка с Алисой.

### 🔄 Не дублирует вкладки (#1)
При вызове через адресную строку или контекстное меню расширение **переиспользует уже открытую вкладку** с Алисой, а не создаёт новую.

### Дополнительно
- **Ctrl+Shift+A** — открыть popup из любой вкладки
- **Выделите текст** → правый клик → **«Спросить у Алисы»**
- **Esc** — закрыть popup
- **Shift+Enter** — перенос строки в поле ввода
- **Тёмная тема** — автоматически подстраивается под систему

## Как это работает

### Режим "Ответ в popup" (по Enter в popup)
```
popup → port → background → hidden tab → alice.yandex.ru
                                            ↓
                                     content.js:
                                     1. ждёт поле ввода
                                     2. вставляет текст
                                     3. нажимает Отправить
                                     4. ждёт стабилизации ответа (1.5 сек)
                                     5. извлекает текст из DOM
                                            ↓
popup ← port ← background ← {response} ←
```
Вся магия происходит за ~3-10 секунд, пользователь остаётся на текущей странице.

### Режим "Открыть в Алисе" (из адресной строки / контекстного меню)
Использует официальный `alice_deeplink` URL-параметр Яндекса. Переиспользует существующую вкладку.

## Установка

### Chrome / Edge / Opera
1. Клонируйте репозиторий
2. Откройте **chrome://extensions/**
3. Включите **«Режим разработчика»**
4. **«Загрузить распакованное расширение»** → выберите папку проекта
5. После обновления версии нажмите **🔄 Обновить** на карточке расширения

## Технические детали

### Коммуникация (для #6)

```
Popup ←→ Port (chrome.runtime.connect) ←→ Background
                                              ↕
                                      Hidden Tab (chrome.tabs.create active:false)
                                              ↕
                                      Content Script (content.js)
```

### Omnibox (#7)

```json
"omnibox": { "keyword": "alice" }
```

В адресной строке: `alice <текст>` → Enter → открывается/обновляется вкладка Алисы.

### Reuse tab (#1)

`openOrReuseTab()` ищет открытые вкладки с `yandex.ru/alice*` и обновляет URL существующей вместо создания новой. Применяется для omnibox и контекстного меню.

### Извлечение ответа (content.js)

После отправки запроса скрипт каждые 250ms проверяет DOM на появление новых элементов, похожих на сообщения ассистента. Когда текст ответа перестаёт меняться в течение 1.5 секунд (ответ завершён), извлечённый текст отправляется в background.

**Если ответ не появился за 20 секунд** — отправляется ошибка.

### Разрешения

- `storage` — история запросов, последний ответ
- `contextMenus` — контекстное меню
- `host_permissions` — только `yandex.ru/alice` и `alice.yandex.ru`
- Никаких данных не собирается и не передаётся
